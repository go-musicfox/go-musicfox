# Quick Task 260715-001: 桌面歌词窗口动态宽度

## Changes
- `internal/desktop_lyrics/desktop_lyrics_darwin.go`

### 修改 1: 减小最小窗口宽度
`minWinW = fontSize * 20` → `minWinW = fontSize * 5`
24pt 字号下最小宽度从 480pt 降到 120pt，短歌词不再留大面积空白。

### 修改 2: updateScrollNeed 支持缩小窗口
原来只放大（`neededWinW > currentWinW`），现在改用 `max(min(neededWinW, maxWinW), minWinW)`，既放大也缩小。

### 关于底部右对齐（已验证无需修改）
posSecond 的 label 填充整个窗口宽度（减 padding），通过 `SetAlignment(1)` 实现右对齐。窗口宽度变化时文本始终在右侧，无需额外修改。

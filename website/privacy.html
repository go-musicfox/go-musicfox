隐私政策

最后更新：2026 年 7 月 7 日

go-musicfox（以下简称"我们"）重视您的隐私。本隐私政策说明当您访问本官网（musicfox.anhoder.com）时，我们如何收集、使用和保护您的信息。

我们收集的信息

本网站使用 Google Analytics（谷歌分析）来了解访问者的使用情况。Google Analytics 会收集以下信息：

  • 访问数据：您访问的页面、停留时间、点击行为
  • 设备信息：浏览器类型、操作系统、屏幕分辨率
  • 位置信息：大致的城市级别地理位置（基于 IP 地址）
  • 引用来源：您从哪个网站或渠道来到本站

这些信息是匿名的，无法用于识别具体个人身份。Google Analytics 使用 Cookie 来区分不同的访问者。Google 的隐私政策见 policies.google.com/privacy。

我们不收集的信息

我们不会收集您的姓名、邮箱地址、登录凭据等个人身份信息。本网站没有用户注册、评论或表单提交功能。

Cookie 使用

Google Analytics 使用第一方 Cookie（如 _ga、_gid）来区分独立用户和跟踪使用模式。这些 Cookie 不包含个人身份信息。您可以通过浏览器设置禁用 Cookie，但这可能会影响分析数据的完整性。

数据存储与保留

Google Analytics 数据的保留期限为 26 个月。我们不会将您的数据存储在自己的服务器上。所有分析数据由 Google 按照其隐私政策处理。

第三方服务

本网站使用了以下第三方服务：

  • Google Analytics — 访问分析
  • Google Fonts — 字体加载
  • GitHub — 项目托管（外部链接）

您的权利

根据适用的数据保护法律（包括 GDPR），您拥有以下权利：

  • 了解我们收集了哪些关于您的数据
  • 要求删除您的数据
  • 反对数据处理
  • 数据可移植性

由于我们收集的数据是匿名的且不由我们直接存储，这些权利主要通过 Google Analytics 的 浏览器插件 来实现。

安全

本网站通过 Cloudflare Pages 托管，所有通信均使用 HTTPS 加密。Cloudflare 的安全性措施见 cloudflare.com/security。

政策更新

本隐私政策可能不时更新。重大变更时，我们会在网站上发布通知。继续使用本网站即表示您同意本政策的最新版本。

联系方式

如果您对本隐私政策有任何疑问，请通过 GitHub Issues 联系我们。

不止是播放器

键盘驱动的完整音乐体验——搜索、发现、管理，一切都在指尖。

⊞

多引擎播放

Beep、MPV、MPD、DLNA、AVFoundation、MediaPlayer——六套引擎按需切换，兼容几乎全部音频格式。从纯 Go 零依赖到系统原生框架，总有一套适合你的场景。

♪

逐字同步歌词

LRC / YRC 逐字时间轴解析，simple / smooth / wave / glow 四种视觉渲染模式。macOS 配合 LyricsX 实现浮动桌面歌词，听歌识曲从未如此沉浸。

⌘

系统深度集成

Linux MPRIS、macOS Now Playing 与菜单栏、Windows 媒体控件全平台覆盖。蓝牙断连自动暂停、系统休眠自动响应，与操作系统无缝协作。

⇄

智能播放模式

列表循环 / 顺序播放 / 单曲循环 / 随机播放 / 无限随机 / 心动模式——六种播放逻辑，一个快捷键循环切换。适配不同听歌心境。

◈

8 级音质解锁

标准品质到鲸云臻享，八档逐级提升。UNM 多源智能回退（酷我/酷狗/咪咕/QQ 音乐），灰色歌曲也能找到可用音源播放。

⌨

全键位可定制

40+ 内置操作全部支持自由重映射。全局快捷键 + TOML 配置，Vim 式 hjkl 导航，纯键盘流操作。鼠标支持单击/双击/滚轮/右键。

◉

Last.fm 记录

自动 scrobble 听歌历史到 Last.fm，可配置记录阈值防止短时切歌污染。保持你的音乐社交网络和 Scrobble 数据始终同步。

▣

Kitty 图形协议

终端内渲染真图形专辑封面——圆角裁剪、旋转动画。双列布局配合鼠标操作，TUI 颜值巅峰。Kitty、iTerm2 等终端原生支持。

↗

自定义分享

Go text/template 驱动分享引擎。歌曲、专辑、歌手、歌单——自由定制模板，一键输出到剪贴板或文件。让音乐分享带上你的个性烙印。

为什么在终端里听歌？

不是因为怀旧——是因为终端就是最高效的人机界面。

01 / 纯粹

零干扰的音乐体验

没有弹窗、没有广告、没有算法推荐的焦虑。没有 Electron 的臃肿，没有浏览器内核的负担。只有你和音乐，以及一个 80 列的终端窗口。极简主义者的完美选择。

02 / 高效

全键盘操作，手不离键盘

Vim 式 hjkl 导航，40+ 操作全部支持自定义快捷键。切歌、搜歌、管理播放列表——一切都在指尖完成。一旦形成肌肉记忆，闭着眼睛也能流畅操作。

03 / 轻量

Go 原生编译，单二进制分发

无 Electron，无浏览器内核。一个数 MB 的单二进制文件，内存占用极低。树莓派、NAS、VPS、老旧笔记本——任何设备都能流畅运行。CPU 占用在日常使用中几乎可以忽略。

04 / 自由

开源 MIT，完全透明

代码完全开放，MIT 许可证。你可以审查每一行代码、修改功能、二次分发。不依赖任何云服务，你的听歌数据和隐私只属于你自己。

六大播放引擎

每个平台都有最优解——也可以纯粹因为偏好选择一个。

>_

Beep

全平台 · 默认引擎

纯 Go 实现，零外部依赖。内置支持 MP3、FLAC、OGG、WAV 格式解码。开箱即用，无需安装任何额外组件。

⎘

MPV

全平台 · IPC 控制

通过 mpv 的 IPC 接口控制播放。支持几乎所有音频格式，性能卓越。适合已使用 mpv 的用户，享受其强大的音频处理管线。

⌂

DLNA

全平台 · 投送引擎

将音乐投送到智能电视、Hi-Fi 音响、游戏机等 DLNA 兼容设备。家庭音乐中心的不二之选。

♫

MPD

Linux · 远程服务

连接本地或远程 MPD 服务器。完美适配 NAS、树莓派等无头音乐服务器场景，多个设备共享一个播放队列。

AVFoundation

macOS · 原生引擎

调用 macOS 原生音频框架，系统级集成最佳。支持音频路由自动切换（蓝牙耳机、外放）。配合 Now Playing 和菜单栏，体验如原生应用般顺滑。

⊞

MediaPlayer

Windows · WinRT

基于 Windows Runtime API，与系统媒体控件、通知中心和任务栏深度集成。原生 Windows 体验。

# 鼠标事件

## 左键 (MouseButtonLeft)

### 歌曲信息行点击
- **播放模式区域**：切换播放模式
- **歌手名称区域**：进入当前播放歌曲的歌手详情页
- **播放/暂停状态区域（图标/文本）**：单击切换播放/暂停
- **心形喜欢区域**：单击切换 喜欢/取消喜欢
- **歌曲名区域**：单击对当前歌曲执行打开M(对于当前播放的操作)

### 菜单项点击
- **单击**：切换焦点到点击的菜单项
- **双击**：播放选中歌曲或进入子菜单
  - macOS (darwin): 400ms 间隔
  - Windows: 500ms 间隔
  - 其他系统: 300ms 间隔

### 进度条操作（窗口最后一行）
- 点击或拖动进度条跳转到指定播放位置
- 如处于暂停状态会自动恢复播放

## 滚轮操作

- **滚轮向下 (WheelDown)**：降低音量
  - Ctrl + 滚轮向下：按配置步长降低音量
- **滚轮向上 (WheelUp)**：增加音量
  - Ctrl + 滚轮向上：按配置步长增加音量

## 右键 (MouseButtonRight)

根据点击位置智能判断行为：

- **歌曲信息行**：打开当前播放歌曲的操作菜单
- **菜单区域**：
  - 先切换焦点到点击项
  - 如果是歌曲菜单，打开选中项的操作菜单
  - 如果是非歌曲菜单，仅切换焦点
- **其他区域**：打开当前播放歌曲的操作菜单

## 中键 (MouseButtonMiddle)

- **点击中键**：返回上一级菜单（等同于 ESC 键）

## 侧键

- **Forward（后退键）**：菜单上一页
- **Backward（前进键）**：菜单下一页
> *注：侧键功能可能受限于终端模拟器支持情况*
